### Pipeline Stages

**`extract()`** — Generates 50 rows of synthetic CSV data with fields `id`, `name`, `category`, `region`, `value`, `quantity`, `score`, writes to `/shared/raw_data.csv`, calls `volume.commit()`.

**`transform()`** — Reads `/shared/raw_data.csv`, uppercases all string column values, adds `revenue = value × quantity` and `score_normalised` columns plus a `pipeline_id` lineage column, writes to `/shared/transformed_data.csv`.

**`load()`** — Reads `/shared/transformed_data.csv`, computes aggregate statistics (min/max/mean/sum per numeric column, revenue grouped by category and region, top-5 by revenue), writes a structured JSON report to `/shared/final_report.json`.
