## What was built (`/home/user/embedding_service/app.py`)

All four Modal capabilities are integrated:

| Capability | Implementation |
|---|---|
| **Modal Volume** | `modal.Volume.from_name("embedding-model-cache", create_if_missing=True)` mounted at `/model-cache`; `volume.commit()` persists weights after first download |
| **Modal Class + GPU + warm pool** | `@app.cls(gpu="T4", min_containers=1)` with a custom `debian_slim` image installing `sentence-transformers==3.0.1` + `torch` |
| **FastAPI web endpoint** | `@modal.asgi_app()` exposes `POST /embed` (accepts `text: str | list[str]`, returns `embeddings` + `model`), plus `/health` and `/stats` |
| **Modal Dict** | `modal.Dict.from_name("embedding-service-stats", create_if_missing=True)` increments `total_requests` and `total_tokens` on every successful call |
