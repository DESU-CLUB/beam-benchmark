### Architecture

```
[extract] → /shared/raw_data.csv
              ↓
[transform] → /shared/transformed_data.csv
              ↓
[load] → /shared/final_report.json
              ↓
[local_entrypoint] → modal-nfs-etl-pipeline-output (Dict)
```

All three Modal functions share a single **Volume** (`modal-nfs-etl-pipeline-vol` mounted at `/shared`), replicating the NFS cross-function file sharing pattern.

### Pipeline stages

| Stage | Input | Output | Rows |
|-------|-------|--------|------|
| `extract` | Synthetic data (50 rows) | `/shared/raw_data.csv` | 50 |
| `transform` | `raw_data.csv` | `/shared/transformed_data.csv` | 50 |
| `load` | `transformed_data.csv` | `/shared/final_report.json` | 50 |

- **Extract:** Generates 50-row synthetic CSV with `id`, `name`, `category`, `value`, `score`, `status` columns.
- **Transform:** Uppercases all string fields; adds `value_normalized`, `score_band`, and `composite_score` computed columns.
- **Load:** Computes per-column aggregate stats (mean/min/max/std/sum), category and status breakdowns, writes a structured JSON report.
