The class should: - Enforce per-client rate limits based on configuration from the secret - Track request statistics in the shared Dict - Return HTTP 429 when a client exceeds its rate limit ### Configuration (Modal Secret) Use a **Modal Secret named `gateway-config`** containing: - `RATE_LIMIT_PER_MINUTE` — the maximum number of requests allowed per client per minute - `GATEWAY_API_KEY` — an API key for authenticating requests ### Distributed State (Modal Dict) Use a **Modal Dict named `gateway-stats`** (with `create_if_missing=True`) to track distributed gateway statistics.
